使用vue ui
vue-cli安装

npm install vue-cli -g
vue-cli的版本查看

vue -V
vue-cli的3.0+以后使用的不是vue-cli了，如果用以上的安装命令安装的并不是最新版的3.0+的，而如果安装3.0的话就需要使用新的

npm install @vue/cli -g
如果原来已经安装了vue-cli的话需要先卸载原来的安装

npm uninstall vue-cli -g
